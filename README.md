# marketing_analytics_system
Marketing Analytics System for Retail Companies in Hong Kong 

## Project Overview
A web-based system built with **Python Django** to help retail companies analyze market data.  
Features include data analysis, dashboards, user login, and report export.

## Minimum Requirements
- **CPU**: Apple M2 Pro / Intel Core i9  
- **RAM**: 3GB+  
- **OS**: Windows 11 (64-bit) or macOS 14.0+  
- **Storage**: 1GB+  
- **Browser**: Latest Google Chrome

## Tech Stack: Python, Django, pandas, MySQL.

## Feature List: Data analysis, dashboard, user login, report export.

## Demo Screenshots: Dashboard, data tables.

## Installation
1. Download and install [Visual Studio Code](https://code.visualstudio.com/).  
2. Download this GitHub Repository, marketing_analytics_system, unzip and open in VS Code.  
3. Install Django extension from Marketplace.  
4. Open a terminal from VSCode and install dependencies:
   ```bash
   python -m pip install Django
   pip install matplotlib pandas seaborn
5. Create a virtual environment and run the server:
   ```bash
   cd myproject
   source .venv/bin/activate
   python -m pip install -r requirements.txt
   python manage.py runserver
7. Open http://localhost:8000 or http://127.0.0.1:8000 in Chrome.
